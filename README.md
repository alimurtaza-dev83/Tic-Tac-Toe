# Tic Tac Toe v1.0

## Overview
A fully functional two-player Tic-Tac-Toe game built using Python and Tkinter.

This project features a clean interface with persistent score tracking across rounds, allowing players to compete in multiple matches without resetting progress.

## Current Features
*   **Turn Management**: Red (X) always goes first for consistent gameplay.
*   **Win Detection**: Automatically detects wins for rows, columns, and diagonals.
*   **Score Tracking**: Points are saved and displayed across multiple rounds.
*   **Restart Option**: Players can reset the board instantly while keeping scores intact.

## Future Features
*   **AI Opponent**: A basic computer player for single-player mode.
*   **Profile**: Players can make unique profiles to be ranked on the leaderboard locally.
*   **Career**: Keeps track of a player's matches, win ratio and rank locally.

## 🛠️ How to Run
1. Install [Python](https://www.python.org/).
2. Download the `tictactoe.py` file.
3. Run the script: `python tictactoe.py`
